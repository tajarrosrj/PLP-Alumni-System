# PLP Alumni System (Web-Based with Database)

## Project Overview
This is a **simple web-based alumni management system** built using **HTML**, **CSS**, **Bootstrap**, **JavaScript**, and **PHP**. It was created as a **final project** for our **Open Source Programming with Database** subject in **3rd year, 1st semester**. The primary focus of the project is to **manage alumni records, profiles, and information** through a **centralized platform with frontend and backend functionality**.

While the system serves its purpose for educational use, **additional features and improvements** may still be added in future versions.

---

## Technologies Used
The project is built using the following technologies:
- **HTML**: For the structure and layout of the system.
- **CSS**: For styling and design.
- **Bootstrap**: For responsive layout and UI components.
- **JavaScript**: For interactivity and client-side behavior.
- **PHP**: For server-side/backend functionality.
- **MySQL**: For database management and storage.
